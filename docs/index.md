# mTOR-NEXUS

mTOR-NEXUS is a provenance-first interactive atlas of mTOR signaling and a
foundation for AI-assisted drug discovery.

Phase 1 provides the heterogeneous graph schema, a curated seed graph, evidence
governance, stable-identifier checks, and reproducible build infrastructure.
Predictive models and broader ingestion workflows arrive in later phases.

!!! warning "Research use only"
    mTOR-NEXUS is not a clinical decision support system.
