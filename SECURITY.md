# Security Policy

## Reporting a vulnerability

Please do not open public issues for suspected vulnerabilities or private
conduct reports. Email the repository owner through the contact address listed
on the GitHub profile for `lysyloxidase`. Include reproduction steps and the
affected version. You should receive an acknowledgement within seven days.

## Scope

The Phase 1 seed graph is research software and is not a clinical decision
support system. Report data integrity concerns through the citation-update
issue template unless disclosure would create a security risk.
